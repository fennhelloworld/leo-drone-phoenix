## ✈️ 飞行前检查清单 (15项)

| # | 检查项 | 标准要求 | 检查方法 |
|---|--------|----------|----------|
| 1 | **天气条件** | 风速<5级, 无雨, 能见度>3km | 天气APP + 现场观察 |
| 2 | **飞行区域** | 非禁飞区, 距机场>8km | 禁飞区地图 + APP查询 |
| 3 | **电池电压** | ≥4.1V/cell (16.4V/4S) | 万用表/电压指示器 |
| 4 | **螺旋桨** | 无裂纹, 紧固, 方向正确 | 目视 + 手动检查 |
| 5 | **电机** | 旋转方向正确, 无异响 | 低速旋转测试 |
| 6 | **结构完整性** | 机架无裂纹, 螺丝紧固 | 目视 + 摇晃测试 |
| 7 | **遥控信号** | 信号强度>90%, 响应及时 | 摇杆响应测试 |
| 8 | **GPS锁定** | ≥8颗卫星, HDOP<2 | QGC查看 |
| 9 | **罗盘** | 校准有效, 偏差<5% | QGC查看 |
| 10 | **数传连接** | 遥测数据正常显示 | QGC连接测试 |
| 11 | **伴飞电脑** | 进程运行, CPU/内存正常 | SSH检查 |
| 12 | **相机画面** | 4路画面清晰, 无遮挡 | Web界面查看 |
| 13 | **Failsafe参数** | 失联返航, 低电返航已设置 | QGC参数检查 |
| 14 | **地理围栏** | 围栏范围和高度正确 | QGC查看 |
| 15 | **安全区域** | 50m内无人员, 降落点清空 | 现场观察 |

## 📡 飞行中监控 (10项)

| # | 监控项 | 正常范围 | 异常处理 |
|---|--------|----------|----------|
| 1 | **飞行高度** | <120m AGL | 接近限制时降低高度 |
| 2 | **水平距离** | <500m (VLOS) | 接近限制时飞回 |
| 3 | **电池电压** | >14.0V (4S) | <14.8V准备返航, <14.0V立即返航 |
| 4 | **信号强度** | >70% | <50%向回飞, 丢失按Failsafe |
| 5 | **GPS卫星** | >8颗 | <6切换视觉定位 |
| 6 | **电机温度** | <80°C | >70°C降低功率, >80°C降落 |
| 7 | **飞行模式** | 符合预期 | 意外切换检查原因 |
| 8 | **位置漂移** | <2m | >5m切换手动控制 |
| 9 | **障碍物** | >3m距离 | <2m触发避障 |
| 10 | **气象变化** | 风速<8m/s | 风速增大立即返航 |

## 🚨 紧急处理程序 (8种场景)

### 场景1: 信号丢失
```
触发: 遥控信号丢失 > 5秒
自动: Failsafe → 返航 (RTL)
手动: 等待自动返航 / 移近恢复信号
注意: 如GPS也丢失, 将原地降落
```

### 场景2: 低电量
```
触发: 电池电压 < 14.8V (20%)
警告: QGC发出低电告警
行动:
  - 20% (14.8V): 准备返航
  - 15% (14.4V): 自动返航
  - 10% (14.0V): 自动降落
注意: 低温环境电池电压下降更快
```

### 场景3: GPS丢失
```
触发: GPS卫星数 < 6 持续 > 5秒
自动: 切换到视觉定位 (如有)
手动:
  1. 切换到Stabilized模式
  2. 目视操控返航
  3. 保持低速
注意: 无GPS时禁止自动任务
```

### 场景4: 电机故障
```
触发: 电机异常转速/停转
自动: Failsafe → 紧急降落
手动:
  1. 立即降低高度
  2. 向安全区域滑翔
  3. 如高度<3m, 紧急断电
注意: 四轴单电机失效无法飞行
```

### 场景5: 伴飞电脑宕机
```
触发: 伴飞无响应 > 3秒
自动: 飞控独立运行
手动:
  1. 切换到手动/Stabilized模式
  2. 使用遥控操控
  3. 正常返航降落
注意: 飞控有独立Failsafe，不依赖伴飞
```

### 场景6: 地理围栏触发
```
触发: 飞出围栏范围
自动: 停止外飞 / 自动返回围栏内
手动:
  1. 确认当前位置
  2. 手动向内飞行
  3. 检查围栏设置
注意: 围栏是最后安全屏障，不要禁用
```

### 场景7: 遭遇强风/恶劣天气
```
触发: 风速 > 8m/s / 降雨 / 能见度下降
行动:
  1. 立即降低高度
  2. 以最低安全速度返航
  3. 迎风降落
注意: 大风时降落需减小降落速度
```

### 场景8: 人员接近
```
触发: 检测到人员进入安全半径 (< 10m)
行动:
  1. 立即停止移动
  2. 上升到安全高度 (>5m)
  3. 悬停等待
  4. 如人员持续接近，飞离
注意: 永远不在人群上方飞行
```

## 🔋 电池安全

### 充电安全
| 规则 | 要求 |
|------|------|
| 充电器 | 必须使用平衡充电器 |
| 充电速率 | ≤ 1C (4A for 4000mAh) |
| 充电电压 | 4.2V/cell (16.8V/4S) |
| 充电温度 | 10°C - 30°C |
| 充电监控 | 全程有人看管 |
| 充电位置 | 防火表面，远离易燃物 |

### 使用安全
| 规则 | 要求 |
|------|------|
| 最低电压 | 3.5V/cell (14.0V/4S) 放电截止 |
| 最大电流 | 50A 持续 (12.5C) |
| 温度范围 | -10°C - 50°C |
| 碰撞后 | 检查电池是否鼓包/损坏 |
| 浸水后 | 立即废弃，禁止使用 |

### 存储安全
| 规则 | 要求 |
|------|------|
| 存储电压 | 3.8V/cell (15.2V/4S) |
| 存储温度 | 10°C - 25°C |
| 存储容器 | LiPo防火袋/防爆箱 |
| 存放位置 | 远离易燃物 |
| 检查频率 | 每月检查电压 |

### 电池退役标准
- ✅ 外壳鼓包 → **立即退役**
- ✅ 容量低于标称80% → 退役
- ✅ 内阻增大超过20% → 退役
- ✅ 循环次数超过200次 → 考虑退役
- ✅ 充电不均衡 (cell差异>0.1V) → 退役
- ✅ 物理损伤 → 立即退役

## 🛡️ Failsafe 配置

### PX4 Failsafe 参数

| 参数 | 值 | 说明 |
|------|------|------|
| `COM_RCL_EXCEPT` | 4 | 遥控丢失后等待秒数 |
| `NAV_RCL_ACT` | 2 | 遥控丢失动作: 返航 |
| `COM_LOW_BAT_ACT` | 2 | 低电动作: 返航 |
| `BAT_CRIT_THR` | 0.10 | 临界电量: 10% |
| `BAT_EMERGEN_THR` | 0.05 | 紧急电量: 5% |
| `NAV_GPSF_LT` | 10 | GPS丢失超时: 10s |
| `NAV_GPSF_ACT` | 3 | GPS丢失动作: 降落 |
| `GF_ACTION` | 2 | 围栏动作: 返航 |
| `GF_MAX_HOR_DIST` | 300 | 水平围栏: 300m |
| `GF_MAX_VER_DIST` | 100 | 垂直围栏: 100m |
| `COM_OBL_ACT` | 5 | Offboard丢失: 降落 |
| `COM_OBL_RC_ACT` | 5 | Offboard+RC丢失: 降落 |

### Failsafe 优先级

```
1. 低电量 (最高优先级)
2. 地理围栏
3. 遥控丢失
4. GPS丢失
5. Offboard模式丢失
6. 避障触发
7. 气象预警
```
